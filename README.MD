Task:
Написать скрипт на Python, который выполняет бэкап файлов конфигураций базовых станций Иркутска из ftp сервера.
Decision:
для работы с проектом использую скрипты bat. 
Необходимо скачать Python (У меня 3.10) и установить ее. Затем уже использовать скрипты:
- checkPython.bat для проверки Python.
- checkBackupFiles.bat для настройки переменного окружения и создания проекта Backup. Скрипт попросит назвать проект, сохранит в домашнюю директорию папку envBackupFiles для работы с переменым окружения, добавит для проекта необходимые библиотеки записанные в файле requirements.txt, из файла config.txt добавит параметры прокси и сохранит папку с вашим названием проекта, откуда будет запускаться проект.
- startBackupFilesFtp.ps1 запускает скрипт. Скрипт запускается в проекте расположенный в домашней папке. в файле config.txt, нужно добавить свои параметры ftp сервера. Скрипт запуускает проект backupFilesFtp.py. в Log файл записывает процедуру запуска. В планировщике задач можно настроить скрипт startBackupFilesFtp.ps1, чтобы он запускался по расписанию.

Decision:
В папке projects/regionsNokiaFTP по отдельности для макрорегионов лежат скрипты mrbtsbackupREG.ps1. Скрипт запускается без зависимости от python. его можно запускать через планировщик задач. Также прикладывается инструкция для работы со скриптом. необохимо расшарить папку, в которой находится скрипт.
так же прикладывается скрипт backupFilesFtp.bat, аналогичные действия делает что и mrbtsbackup.ps1.
